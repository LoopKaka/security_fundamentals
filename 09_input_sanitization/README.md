# 09 Input Sanitization

Simple HTML demo that shows how unsafe DOM insertion can render user-provided input as HTML.

## Prerequisites

- Visual Studio Code
- Live Server extension

## How to Run

1. Open this folder in VS Code.
2. Right-click `home.html`.
3. Select `Open with Live Server`.
4. Open the Live Server URL in your browser. It is usually:

   ```text
   http://127.0.0.1:5500/09_input_sanitization/home.html
   ```

## Endpoints

This is not a Node/Express project. It has no API endpoints.
