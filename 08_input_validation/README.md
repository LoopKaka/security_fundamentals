# 08 Input Validation

Simple Express app that serves a form from `public/home.html` and validates form submissions on the server.

## Prerequisites

- Node.js
- npm

## How to Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm start
   ```

3. Open the app in your browser:

   ```text
   http://localhost:2000
   ```

The server runs on port `2000`. Press `Ctrl+C` in the terminal to stop it.
